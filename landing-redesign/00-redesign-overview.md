## Project Scope

Redesign of the primary Captain dashboard (cookbook main page) for Small Plates & Co. desktop platform.

**In Scope:**
- Recipe visualization and list
- Add recipe flow (Captain-side)
- Member management (invite captains)
- Core navigation and information architecture
- Visual identity application (color, typography, spacing)
- Micro-copy and voice alignment

**Out of Scope (for now):**
- Share link flow (Collection link - Get Plates)
- Guest submission experience
- Mobile adaptation
- Print Book flow

## Design Principles (derived from Brand Book)

### The Margot Cole Test

Before any design decision, ask:

| Question | Required Answer |
|----------|-----------------|
| Would Margot be frustrated in the first 2 minutes? | No |
| Does this look like it could be any other SaaS? | No |
| Would she put this on her screen with pride? | Yes |
| Does every word earn its place? | Yes |
| Does this feel made with care? | Yes |

### Visual Identity Application

| Element | Specification |
|---------|---------------|
| **Primary Background** | Warm White #FAF7F2 |
| **Primary Accent (CTAs)** | Honey #D4A854 |
| **Primary Text** | Soft Charcoal #2D2D2D |
| **Secondary Text** | Warm Gray #9A9590 |
| **Cards/Containers** | Sand #E8E0D5 or Cream #F5F1EB |
| **Headlines** | Minion Pro (or editorial serif) |
| **Body/UI** | Inter |

### Voice Principles

- Cool on the outside, emotional on the inside
- Direct, not blunt
- No exclamation marks (almost never)
- Sound certain, not eager
