# 1. EL PROYECTO

## ¿Qué es Small Plates?

Small Plates crea **libros de recetas colaborativos para bodas**. Los invitados contribuyen recetas a través de nuestra plataforma, y nosotros las transformamos en un libro de cocina de tapa dura para la pareja.

**El producto:** Un libro de recetas hecho por los invitados, regalado a los novios.

**La promesa:** "Recipes from the people who love you."

## ¿Por qué necesitamos un sistema?

Cada libro es **único** — diferentes invitados, recetas e historias. No existe un libro igual a otro.

Esto significa que necesitamos un **sistema de diseño escalable**, no solo un libro bonito. Cada libro debe sentirse premium, pero debe poder producirse eficientemente.

# 2. LA VISIÓN

## Principio rector

> **"Cool on the outside. Emotional on the inside."**

El libro debe verse sofisticado y editorial. Pero cuando lo abres, debe conmover.

## El filtro de diseño

> *"¿Una mujer de 28 años, sofisticada, con buen gusto, querría este libro en su cocina?"*

# 3. EL JOURNEY EMOCIONAL

El libro cuenta una historia en tres actos. **Esta curva emocional es fundamental para entender lo que buscamos:**

```
INTENSIDAD
EMOCIONAL
    ▲
    │                                                    ★ "Still at the table"
    │                                                   /
    │                                                  /
    │           Comprensión                          /
    │              /\                               /
    │             /  \                             /
    │    Intriga /    \                           /
    │          /       \      [RECETAS]         /   Cierre
    │         /         \    Plateau de       /    emocional
    │  Calma /           \   conexión        /
    │  Sage  \            \  ───────────────/
    │   │     \            \               /
    └───┼──────────────────────────────────────────────────► PÁGINAS
      Pág 1   Front Matter      Recetas       Back Matter   Pág Final
      
TONO:   Cool ──────────────> Cálido ────────> Profundo ───> Cool
```

**ACTO I (Front Matter):** El libro se presenta con confianza. No se explica, intriga.

**ACTO II (Recetas):** Cada receta es una persona. El lector conecta con los contribuyentes.

**ACTO III (Back Matter):** El momento emocional. Aquí ganamos las lágrimas. "Still at the table."

### Lectura de la curva

| Momento | Emoción | Descripción |
|---------|---------|-------------|
| Página 1 | Cool | Silencio. Color sólido. Sin explicación. |
| Front Matter | Ascenso gradual | Imágenes intrigantes, luego comprensión del concepto. |
| Recetas | Plateau cálido | Conexión constante, pero ninguna es el clímax. |
| Back Matter | Ascenso hacia clímax | Reflexión, articulación, culminación. |
| "Still at the table" | **Pico** | El momento más emocional. Todo construyó hacia aquí. |
| Cierre | Descenso a cool | Silencio post-clímax. Volvemos al Sage. Elegancia. |

# 5. DISEÑO DE PORTADA

## Dirección creativa

Buscamos una portada **moderna, minimalista y editorial**. Debe verse como un objeto de diseño que la pareja estaría orgullosa de tener como adorno de mesa en su cocina.

> **El principio:** Cool on the outside, emotional on the inside.
> La portada es "the outside" — debe ser elegante, contenida, sofisticada.

## Referencia visual

**Magnolia Table Vol. 2** de Joanna Gaines es nuestra referencia principal:
- Fondo claro/blanco
- Mucho espacio
- Tipografía editorial elegante
- Se siente "de boda" sin ser cursi
- Transmite calidad y cuidado

## Especificaciones

### Portada frontal

| Elemento | Especificación |

| **Fondo** | Color Beige/Warm White (#FAF7F2 o similar) |

| **Nombres de novios** | En la parte inferior, centrados |
| **Tipografía nombres** | Minion Pro, estilo editorial elegante, Soft Charcoal |
| **Composición** | Mucho espacio en blanco. No saturado. Respiración. |

|----------|----------------|

### Contraportada

| Elemento | Especificación |
|----------|----------------|
| **Fondo** | Mismo tono Beige/Warm White de la portada |
| **Contenido** | Brand statement breve o tagline |
| **Tono** | Simple, sin llamar mucha atención |

Posibles textos para contraportada:

## Nota sobre dirección

En esta etapa, preferimos **no tomar riesgos grandes** con la portada. Buscamos algo seguro, elegante, y universalmente atractivo. El estilo minimalista, blancos, "wedding-style" es intencional — es una posición conservadora mientras validamos el producto.

### Detalle de `M-Main`:

```
┌─────────────────────┬─────────────────────┐
│   PÁGINA IZQUIERDA  │   PÁGINA DERECHA    │
├─────────────────────┼─────────────────────┤
│  <<recipe_name>>    │                     │
│  <<guest_name>>     │    [IMAGEN DE       │
│                     │     LA RECETA]      │
│  <<comments>>       │                     │
│                     │                     │
│  <<ingredients>>    │                     │
│  <<instructions>>   │                     │
│                     ├─────────────────────┤
│  [footer]           │  [recipe_name]  [#] │
└─────────────────────┴─────────────────────┘
```

**Características:**
- Los placeholders `<<campo>>` se conectan a nuestro script de automatización
- El footer derecho muestra automáticamente el nombre de la receta
- Los Paragraph Styles se aplican automáticamente

## Sistema de automatización

Tenemos un **script funcional** que extrae recetas de nuestra base de datos y genera las páginas automáticamente.

**Importante:** Los placeholders `<<recipe_name>>`, `<<guest_name>>`, etc. son necesarios para la automatización. Cualquier cambio debe mantener esta nomenclatura.

# 8. LO QUE NECESITAMOS

## 8.1 Mejoras al template de recetas (`M-Main`)

- Revisión del diseño visual (tipografía, espaciado, jerarquía)
- Detalles editoriales (separadores, ornamentos sutiles)
- Que se sienta "premium"

**Pendiente por resolver — Overflow:**

Cuando una receta no cabe en 1 dupla, necesitamos que continúe:

```
RECETA NORMAL (1 dupla):
┌─────────┬─────────┐
│  TEXTO  │ IMAGEN  │
└─────────┴─────────┘

RECETA LARGA (2 duplas):
┌─────────┬─────────┐     ┌─────────┬─────────┐
│  TEXTO  │  TEXTO  │  +  │ IMAGEN  │ [vacío] │
│ (pt. 1) │ (pt. 2) │     │         │         │
└─────────┴─────────┘     └─────────┴─────────┘
```

## 8.2 Divisores de categoría

Las recetas se organizan por categorías. Necesitamos un template de "Category Divider":

| Categoría |
|-----------|
| Starters & Snacks |
| Soups & Salads |
| Mains |
| Sides |
| Desserts |
| Drinks & Others |

*No todos los libros tendrán todas las categorías.*

## 8.3 Index / Índice

Tenemos un índice funcional pero **no nos gusta visualmente**. Necesitamos rediseño.

## 8.4 Front Matter y Back Matter

Tenemos avance inicial. Necesitamos revisión y refinamiento.

*Las imágenes para todos los fillers las proporcionamos nosotros.*

## 8.5 Diseño de portada

Diseño en dos formatos: **8×10** y **6×9**
