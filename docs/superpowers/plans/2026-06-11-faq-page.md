# FAQ Page (`/faq`) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the dedicated `/faq` page (Remento-style: jump-nav + top-questions accordion + complete FAQ in category cards) per `docs/superpowers/specs/2026-06-11-faq-page-design.md`, plus a "See all FAQs" link on the main landing FAQ section.

**Architecture:** Server component page following the `/pricing` pattern (`Banner theme="light"` + `Footer` + `WhatsAppFAB`). All content lives in one typed data file. One small client component (`JumpNav`) for the sticky sidebar / mobile pills with active-section highlight. Top-questions accordion uses native `<details>/<summary>` — no framer-motion, no JS needed. JSON-LD `FAQPage` schema emitted from the same data.

**Tech Stack:** Next.js 14 App Router, TypeScript strict, Tailwind with brand tokens (`brand-warm-white`, `brand-sand`, `brand-charcoal`, `brand-honey`), `type-*` typography classes from `app/globals.css`.

**Testing note:** This is a static marketing page with no logic worth unit-testing (per project conventions: no test files exist for landing pages). Verification = `npx tsc --noEmit` + `npm run lint` at the end, then visual check via Ricardo's screenshots of the dev server (project rule: no Playwright/headless).

**Voice guardrails for any copy adjustments during execution:** never use cherish / treasure / memories / special / unique / loved ones / celebrate / journey / curated / perfect / amazing. No specific guest counts. The copy in Task 1 was already founder-reviewed — do not rewrite it.

### Task 2: JumpNav client component

**Files:**
- Create: `app/(public)/faq/_components/JumpNav.tsx`

Sticky vertical list on desktop, horizontally scrollable pill row on mobile. Highlights the section in view via IntersectionObserver. Pills use raw `text-sm` — functional navigation UI, allowed exception to the `type-*` rule.

- [ ] **Step 1: Create the component**

```tsx
"use client";

import { useEffect, useState } from "react";

interface JumpNavSection {
  id: string;
  title: string;
}

export default function JumpNav({ sections }: { sections: JumpNavSection[] }) {
  const [activeId, setActiveId] = useState<string>("");

  useEffect(() => {
    // Reason: highlight the category currently in the upper band of the
    // viewport; margins make exactly one section "active" at a time.
    const observer = new IntersectionObserver(
      (entries) => {
        const visible = entries.filter((entry) => entry.isIntersecting);
        if (visible.length > 0) setActiveId(visible[0].target.id);
      },
      { rootMargin: "-15% 0px -75% 0px" }
    );
    sections.forEach(({ id }) => {
      const element = document.getElementById(id);
      if (element) observer.observe(element);
    });
    return () => observer.disconnect();
  }, [sections]);

  return (
    <nav
      aria-label="FAQ sections"
      className="-mx-6 mb-8 overflow-x-auto px-6 lg:sticky lg:top-28 lg:mx-0 lg:mb-0 lg:overflow-visible lg:px-0"
    >
      <p className="type-eyebrow mb-4 hidden lg:block">Jump to a section</p>
      <ul className="flex gap-2 lg:flex-col">
        {sections.map(({ id, title }) => (
          <li key={id} className="shrink-0">
            <a
              href={`#${id}`}
              className={`inline-block whitespace-nowrap rounded-full border px-4 py-2 text-sm transition-colors ${
                activeId === id
                  ? "border-brand-charcoal bg-brand-charcoal text-white"
                  : "border-brand-sand bg-white text-brand-charcoal hover:border-brand-charcoal/40"
              }`}
            >
              {title}
            </a>
          </li>
        ))}
      </ul>
    </nav>
  );
}
```

  title: "FAQ — Small Plates & Co.",

    "Everything about how the book gets made — what's free, what it costs, how guests send recipes, and when it shows up at your door.",

function AnswerBody({ item, className = "" }: { item: FaqItem; className?: string }) {
  return (
    <div className={className}>
      {item.answer.map((paragraph) => (
        <p key={paragraph} className="type-body-small mb-3 last:mb-0">
          {paragraph}
        </p>
      ))}
      {item.list && (
        <ol className="list-decimal space-y-2 pl-5">
          {item.list.map((entry) => (
            <li key={entry} className="type-body-small">
              {entry}
            </li>
          ))}
        </ol>
      )}
    </div>
  );
}

export default function FaqPage() {
  const topQuestions = TOP_QUESTION_IDS
    .map(getFaqItem)
    .filter((item): item is FaqItem => item !== undefined);

  const faqJsonLd = {
    "@context": "https://schema.org",
    "@type": "FAQPage",
    mainEntity: FAQ_CATEGORIES.flatMap((category) => category.items).map((item) => ({
      "@type": "Question",
      name: item.question,
      acceptedAnswer: {
        "@type": "Answer",
        text: [...item.answer, ...(item.list ?? [])].join(" "),
      },
    })),
  };

  return (
    <>
      <Banner theme="light" showShippingStrip={false} />
      <main className="min-h-screen bg-brand-warm-white">
        <script
          type="application/ld+json"
          dangerouslySetInnerHTML={{ __html: JSON.stringify(faqJsonLd) }}
